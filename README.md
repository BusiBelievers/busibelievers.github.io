# Busi Believers Incorporated

This repository hosts the static website for Busi Believers at https://busibelievers.github.io.

## Local preview

Open `index.html` in your browser or use a simple server:

```bash
cd /home/runner/work/busibelievers.github.io/busibelievers.github.io
python -m http.server 8000
```

Then visit http://localhost:8000.

## Deploying

Push changes to the default branch. GitHub Pages will serve the updated `index.html` from the repository root.
